📝 Descripción
Adaptación, limpieza y optimización de una plantilla web para la empresa tecnológica EcoPulse Dynamics, especializada en Inteligencia Artificial aplicada a la energía.

🚀 Modificaciones Realizadas
1. Gestión de la Información
Limpieza de código: Eliminación de secciones redundantes y texto de relleno (Lorem Ipsum).

Actualización de servicios: Implementación de contenido real sobre Smart Grid AI, Solar Modular y V2G.

Localización: Traducción completa al castellano y actualización de datos de contacto (Plasencia, Cáceres).

2. Optimización y Rendimiento (WPO)
Formato WebP: Conversión de todas las imágenes a formato de última generación para reducir el tiempo de carga.

Compresión: Optimización de recursos multimedia mediante Squoosh sin pérdida de calidad visual.

🛠️ Tecnologías utilizadas
HTML5 / CSS3 (Estructura y estilos personalizados).

Bootstrap (Diseño responsive).

JavaScript / jQuery (Efectos y filtrado de proyectos).
